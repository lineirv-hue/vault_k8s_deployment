Setup — Enable Chrome AI
  1. Open chrome://flags/#prompt-api-for-gemini-nano → set Enabled
  2. Open chrome://flags/#optimization-guide-on-device-model → set Enabled BypassPerfRequirement
  3. Go to chrome://components, find Optimization Guide On Device Model, click Check for update
  4. Restart Chrome, then open Google Messages
No remote AI calls — everything runs on-device.
